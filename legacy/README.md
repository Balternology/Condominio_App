# Archivos Legacy

Esta carpeta contiene archivos relacionados con Docker que no se usan en el setup actual con XAMPP/MySQL.

## Archivos

- `docker-compose.yml` - Configuración de Docker Compose (usa PostgreSQL)
- `Dockerfile` - Dockerfile para el frontend
- `backend-Dockerfile` - Dockerfile para el backend
- `start-all.sh` - Script de inicio para Linux/Mac usando Docker

## Nota

El proyecto actual usa:
- **XAMPP** para MySQL
- **Scripts .bat** para Windows
- **Entorno virtual Python** local
- **npm** para el frontend

Si necesitas usar Docker en el futuro, estos archivos están disponibles aquí.
